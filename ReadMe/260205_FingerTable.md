# 요약: 탐색 규칙
현재 노드의 successor 범위 안이면 종료.
아니면 finger table에서 key보다 작은 가장 큰 노드를 선택해 이동.
이동 후 다시 1번 검사.
이 과정을 반복하면 O(log N) 단계 안에 찾을 수 있음.
